- `local-docker-setup.md` - local Docker/Ollama setup and smoke checks.
- `local-benchmark-snapshot.md` - benchmark snapshot for latency/quality/cost context.
- `model-stats-matrix.md` - candidate model matrix and licensing notes.
- `serving-options.md` - serving/provider strategy options and tradeoffs.
- `serving-infra-blueprints.md` - implementation blueprints for self-hosted serving.
- `finetune-model-shortlist.md` - fine-tune shortlist and bakeoff guidance.
- `eval-scorecard-template.md` - evaluation scorecard template.
- `data-provenance-manifest-template.md` - provenance tracking template.
