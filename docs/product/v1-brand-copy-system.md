# Ingrediential V1 Brand and Copy System

This document defines V1 naming and tone for both web and native surfaces.

## Product Name

- Canonical name: `Ingrediential`
- Short descriptor: `ingredient-first cooking studio`

## Voice Rules

- Be direct, practical, and calm.
- Lead with user outcome first, then mechanics.
- Prefer active verbs: `Discover`, `Save`, `Find matches`, `Apply filters`.
- Keep trust language explicit when generated content appears (`generated fallback`, `reviewable`).

## Core Messaging Pillars

- Cook from what you already have.
- Reduce decision friction with ranked matches.
- Keep saved recipes and account sessions reliable.

## Primary Copy Anchors

- Home headline: `Cook great meals from what you already have.`
- Home subheadline: `Ingredient-first search, reliable favorites sync, and fast recipe detail workflows for everyday cooking.`
- Discover CTA: `Find matches`
- Save CTA: `Save Recipe`

## Navigation Labels (V1)

- Web: `Home`, `Recipes`, `Favorites`, `Account` (admin lanes unchanged)
- Mobile: `Discover`, `Saved`, `Profile`

## Error and Recovery Copy Guidelines

- Start with plain-language issue summary.
- Add one next action where possible (`Retry`, `Browse catalog`, `Continue with typed ingredients`).
- Keep auth/session errors concise and action-oriented.
