# Web Visual Direction Board v2

Companion to `design-modernization-v1.md`, updated for Ingrediential V1 quality gates.

## Intent

- Keep recipe operations high-signal with a cleaner, modern visual language.
- Improve scanability and trust on Discover and Detail without over-designing.

## Direction Snapshot

- **Tone:** practical culinary workspace.
- **Typography:** `Space Grotesk` for hierarchy, `Plus Jakarta Sans` for UI body, `IBM Plex Mono` for compact metadata.
- **Color strategy:** cool neutral surfaces with blue action accents and warm highlight overlays.
- **Surface strategy:** layered cards, soft borders, compact chip rails.
- **Motion strategy:** short reveal and subtle hover lift only where it supports interaction clarity.

## Component Guidance

- Shell: consistent sidebar + topbar context with clear product framing (`Ingrediential V1`).
- Search modules: split into `Ingredient-first search` and `Library browser` cards.
- Result cards: keep source, match, and trust metadata in fixed ordering.
- Detail page: section cards for Ingredients/Steps and dedicated analysis score grid.

## Anti-Patterns

- No generic default-dashboard styling.
- No oversized animation in dense result lists.
- No mixed CTA language (`Add Favorite` vs `Save Recipe`) on same surface.
