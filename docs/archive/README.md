# Archive Docs

Historical snapshots of completed checklist work and retired planning groups.

- `checklist-completed-groups.md` - fully completed checklist groups removed from active checklists.
- `checklist-completed-items.md` - completed checklist items moved out of active/open backlog views.
- `mobile/`, `product/`, `llm/`, `deliverables/` - dated snapshots moved out of active doc indexes.

Archive entries should include date and source checklist path.
