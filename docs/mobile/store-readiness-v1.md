# Mobile Store Readiness v1

## Play Store

Required before production submission:

- App icon, feature graphic, and screenshots.
- Data safety form and privacy policy URL.
- Content rating questionnaire completed.
- Closed testing run and pre-launch report reviewed.

## Apple App Store

Required before production submission:

- App icon + screenshots by required device classes.
- App Privacy declarations and policy URLs.
- TestFlight internal/external cycle completion.
- Final metadata + review checklist signoff.

## Asset governance

- Keep source assets in design system workspace.
- Version generated store assets per release candidate.
- Capture final text metadata in release PR notes.
